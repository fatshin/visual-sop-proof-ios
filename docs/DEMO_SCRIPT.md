# Oracle Council demo

## 0:00–0:08 — The problem

Show the opening statement:

> 未来を占う前に、自分の記憶を読む。

Explain that ordinary readings forget the life that came before the question.

## 0:08–0:22 — Import memories

Choose the sample path. Show that two memories came from Obsidian, one from the
ChatGPT Memory summary, and one from a ChatGPT conversation.

Emphasize that assistant messages are not imported as the user's memories.

## 0:22–0:34 — Consent

Turn one memory off and on. Explain that no memory enters the reading until the
user has reviewed and selected it.

## 0:34–0:50 — Ten readings

Ask:

> 次の仕事へ進むために、今どの準備を始めるべきですか

Generate the council. Scroll through I Ching, Four Pillars, Nine Star Ki,
Sukuyō, Western astrology, tarot, numerology, runes, geomancy, and lunar
reflection. Show which specific memory each reading references.

## 0:50–1:00 — Return to reality

Show the grounded next step and save the result as an Obsidian Markdown note.
End on the seven-day review section.

## Closing line

Oracle Council does not predict a fact. It helps you notice what your past self
has already been trying to tell you.
