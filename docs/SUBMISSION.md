# Oracle Council submission brief

## Problem

Divination apps usually give a generic result and forget the user's lived
history. Journaling tools preserve history but rarely help a person revisit it
at the moment of a difficult decision.

## Product

Oracle Council lets a user review and select memories imported from Obsidian,
a ChatGPT Memory summary, or a ChatGPT conversation export. Ten symbolic
traditions independently interpret the same question, and each result cites
the memory it used.

## Trust boundary

- Local processing is the default.
- ChatGPT assistant responses are excluded from memory import.
- Hidden Obsidian configuration and likely secrets are excluded.
- Users approve each memory before interpretation.
- Results are reflective prompts, not factual predictions.
- The exported note asks what happened seven days later.

## Why it matters

The emotional result is not “the app knew the future.” It is “I finally noticed
the pattern I had been writing down for months.”
