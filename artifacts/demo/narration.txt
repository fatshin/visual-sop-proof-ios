Public Notice Stress Test compares a draft notice with a small authoritative fact record before publication. This tested fixture has conflicting dates, amount, age, residency, and contact information, plus one promise that has no supporting source. Its verified fixture result blocks release and returns six precise repairs. Each repair is tied to the source value beside it. I used Codex with GPT-5.6 to define the claim checks, implement the release gate, generate the evaluation, and review the wording for unsupported certainty. A live GPT-5.6 extension can simulate likely misreadings and suggest clearer language, while deterministic fact comparison remains the publication gate.
